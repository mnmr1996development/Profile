  • AI in simple Game

    Project Description:

    The purpose of this project was to demonstrate machine learning practices commonly used today. My college colleauge and I designed a video game similar to asteroids where the user would have to avoid all the incoming rocks and had a neural network figure out how to play the game from scratch.

    Disclaimer: traditionally this would train a lot faster and a lot better but this was done using a windows integrated graphics laptop instead of a tradition GPU, this was the maximum runnable settings before the program got way too slow

    The First 6 iterations demonstartes when the neural network was moving at random

    The 104th iteration demonstartes when it started picking up some patterns

    AT the 643rd generation (timestamp 1:47) the best ship in the generation is able to survive for over 3 minutes
